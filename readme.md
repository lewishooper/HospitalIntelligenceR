This is a consolidation of work done elsewhere.
The key component of this is an extraction framework for getting information from hospital websites
Currently this work is done in other projects with separate approaches to scaping
this will create a single scrapping structure that will be used for different extraction goals/Roles
the current roles are as follows
1. Extract strategic plans once  a year (developed in another project)
2. Extract Executives once a month (developed in another project)
3. Extract Board members Once a year (included in # 2 but not currently useful)
4. Extract Foundational document Once a year.  Mission,Vision Values (not yet developed)

This work will be done in R /Rstudio and will use the claude api
